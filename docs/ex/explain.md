FILE *filePointer;：这行声明了一个指向 FILE 类型的指针变量 filePointer。FILE 类型是C语言中用于处理文件的结构体类型，filePointer 将用于指向文件流。  
~~~

~~~
filePointer = fopen("output.txt", "w");：这行代码打开一个名为 "output.txt" 的文件以便写入（写入模式为 "w"）。fopen() 是一个C标准库函数，用于打开文件并返回一个文件指针。如果成功打开文件，filePointer 将指向这个文件流；否则，如果文件无法打开，filePointer 将为 NULL。

~~~
在 C 语言中，who->name 是使用结构体指针 who 访问结构体成员 name 的方式。

假设有如下的结构体定义：


struct Person {
    char name[50];
    int age;
};

如果你有一个指向 Person 结构体的指针 who，你可以使用箭头操作符 -> 来访问该结构体的成员，而不必使用间接引用操作符 * 和点操作符 .：

rrno:
errno 是一个错误码，在 C 语言中用于指示函数调用或操作是否产生了错误。当某个函数执行出错时，它会设置相应的错误码，开发者可以根据这个值来判断出错的原因。

perror:
perror 是一个函数，用于将由 errno 确定的错误信息输出到标准错误流（stderr）。它输出一个描述错误原因的字符串，以及一个对应 errno 值的错误信息。通常用于在程序中输出相关的错误信息。

fopen:
fopen 是一个标准库函数，用于打开文件，并返回一个文件指针。它接受文件名和打开模式作为参数，根据模式打开文件，并返回一个指向该文件的文件指针。如果打开文件失败，返回 NULL。

fread:
fread 是一个标准库函数，用于从文件中读取数据。它接受数据块的大小、数据块的数量、文件指针以及要读取数据的位置等参数，并将读取的数据存储到指定的内存块中。

fclose:
fclose 是一个标准库函数，用于关闭一个先前打开的文件。它接受一个文件指针作为参数，并将与该文件相关联的资源（如文件描述符）释放，关闭文件。

rewind:
rewind 是一个标准库函数，用于将文件指针重新定位到文件的起始位置。它接受一个文件指针作为参数，并将文件指针定位到文件的开头。

memcpy:
memcpy 是一个标准库函数，用于在内存块之间复制一定数量的字节。它接受目标内存地址、源内存地址以及要复制的字节数作为参数，并将源内存中的数据复制到目标内存中。

atoi:
atoi 是一个标准库函数，用于将字符串转换为整数。它接受一个表示数字的字符串作为参数，并返回对应的整型
